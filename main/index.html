Начало ​

MINECRAFT СНГ – народный ванильный Minecraft сервер для всех желающих.

Начать играть ​

Для начала нужно быть участником нашего Discord сервера.

Далее необходимо перейти в канал #💚・начать и выбрать тип вашего аккаунта (лицензионный или пиратский), после чего заполнить заявку.

ПОДСКАЗКА

Если авторизоваться через наш сайт, то Вас автоматически присоединит к Discord серверу.

✅ Если вашу заявку одобрили

Добавляйте адрес сервера play.minecis.net в список серверов Сетевой игры и подключайтесь, после одобрения заявки Ваш аккаунт автоматически получает доступ к серверу. Поздравляю и добро пожаловать 🥳

play.minecis.net
❌ Если вашу заявку отклонили

А если не хочу?

У Вас всё еще есть возможность повторно подать заявку через 3 дня с момента отклонения. Вы также можете воспользоваться Платным входом.

ПОДСКАЗКА

Платный вход служит гарантией добросовестной игры. Это также помогает упростить получение доступа к серверу одновременно поддержав проект материально

Подключение ​

Наш сервер работает на версии Minecraft 1.21.11 Java Edition, для подключения используйте адрес play.minecis.net .

Приятной игры!
